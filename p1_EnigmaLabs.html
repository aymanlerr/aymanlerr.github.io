EnigmaLabs E-Commerce Website

I created a laptop e-commerce website UI/UX design using Adobe XD. THe website uses a horizontal navigation bar with essential links such as home, view all laptops, search, sign in/up and view card. Instead of taking up all the space at the top, I made the taskbar size to fit the icon-links nicely. Animations are used in each page and most elements. I tried to use a fancy but time-consuming way to animate the checkout payment section
