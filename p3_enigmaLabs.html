EnigmaLabs Website Design

This is a laptop e-commerce website UI/UX design. The website theme is curves edges and futuristic. The website utilizes design principles to further improve the user experience when browsing through the website and securing a purchase.

Key features:

  • Transition, transform effects
  • Item comparison chart
  • Potential filter function

Software:

  • Adobe XD

Home

All of the laptops on sale will be shown in this page. There is a filter function at the top to filter the key specifications of the user's ideal laptop.

All products

This is a snippet of what it will looks like for when a user clicked on a laptop. The user will be able to view the laptop's specifications and price. At the same time, the compare feature and buy button is displayed to allows seamless movement to different pages.

View specific item

Users will be able to compare 2 laptops. The specifications will be compared side-by-side with a 'Add to Cart' button placed directly below each laptop. This allows the user to make a comparison and easily add their ideal laptop to cart.

Product comparison

The cart will not only display the items' name but also display the item's specifications to allow the user to double confirm the laptop that they are going to buy. Net prices are shown at the right side of the screen where it will not overlay on any of the cart items.

Cart

In the payment page, users will be required to enter their payment detail. There is input validation thus if any of the input fields are empty, the payment will not proceed.

Payment

This is the background information of the company. There are free-form designs for the pictures to compliment the website theme.

Our story

Users who want to earn points can register an account for free. The registration simply requires their new username, full name, email, new password and birth date. There is input validation for every input field

Register

If the user has already registered, they can login using their username and password. There is a 'Forgot password' option for those who does not remember their password.

Login

Upon login, users are able to view administration options for their account, order history, rewards and the order tracker. The profile image can be changes to their liking as well.
